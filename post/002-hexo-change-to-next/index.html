【Hexo】Hexo更换为NexT主题

Hexo

hexo默认的主题是 landscape 。

更换主题

下载主题

进入要更换主题的网站项目内，进入 themes 文件夹下，将要更换的主题放置在该目录下。例如：

1
2
3
4
5

# 进入网站项目的根目录下，例如：
cd hexo-test

# 安装next主题，把它下载至themes文件夹下的next文件夹内
git clone <https://github.com/theme-next/hexo-theme-next> themes/next

更换配置

主题下载好之后，编辑 _config.yml 进行更换主题。

找到 theme: landscape 这一行，将默认主题 landscape 改为你要修改的主题，例如： next 。

检查新主题是否生效

1
2
3
4

hexo s

# 如果未生效，可以先执行clean命令后再试
hexo clean

升级主题

1
2
3
4
5

# 进入网站项目的主题文件夹，例如：
cd hexo-test/themes/next

# 更新
git pull
